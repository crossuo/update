.:[ 2019-02-03 ]:.

X:UO Launcher 1.0.1:


New: UI redesign to help new users and avoid configuration issues
New: Protocol Client Version in Advanced tab to use different packet versions from client data version
Fix: Tooltips on fields to help new users
Fix: Temporary download cache data moved to App Data (%APPDATA% on Windows)

CrossUO 1.0.1:


New: 'ProtocolClientVersion' config option to use different packet versions from client data version
New: 'Language' config option to use different language than the system locale (ENU, FRA, etc.)
New: Parity - CTRL+Q to repeat last console text entry
Fix: Text decoding on Linux/MacOS (for languages with special characters)
Fix: Window resizing issue on login/logout. Thanks @Johnny.
Fix: Cryptography issue causing freeze during login
Remove: Ping feature. This required administrator privileges and was a huge source of issues.

.:[ 2019-01-25 ]:.


Welcome to CrossUO, the very first Ultima Online client with real native multiplatform support.

This is the first release of a new client, so expect issues. We need to release this so you can help us with testing and getting the project in shape going forward.
There is zero new features since the last Orion release, do not expect much. Minor updates will follow until we consider this stable enough to start fixing known Orion bugs and new features.

Be patient and report any issues. Thank you!
